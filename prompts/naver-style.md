# 네이버 블로그 스타일 가이드 — 컬컴 하남 (스터디 후기 / 현황)

레퍼런스: prompts/samples/naver/*.md (10개) — 모두 컬컴 **하남점** 글이며, **스터디 후기 / 스터디 현황** 카테고리에 한정.

이 가이드의 범위는 **하남점 한정**, **후기·체험·스터디 현황** 한정이다. 다음은 다루지 않는다:
- 다른 지점 (잠실·분당)
- 학습 콘텐츠(표현 강의, 문법, 회화 팁) 단독 글
- 이벤트·플리마켓 단독 안내 글

## 1. 글의 정체성

- **장르**: 1인칭 후기·체험기·스터디 현황 일기.
- **화자(페르소나)**: 다음 중 사진과 메모에 가장 잘 맞는 하나를 잡는다.
  - 직장인 회원 (해외영업·바쁜 직장인)
  - 아이 엄마 / 영유맘 / 7세 학부모
  - 왕초보 / 영어 울렁증
  - 장기 회원 (6개월~1년차) "달라진 점"
  - 컬컴 조교 시점 ("스터디 분위기를 이끌고 있어요")
- **목표**: 컬컴 하남의 분위기를 1인칭으로 보여주고, 마지막에 카카오톡 문의로 자연스럽게 유도.

## 2. 길이와 골격

- **분량**: 본문 1,800~3,500자 (제목·매장 박스·해시태그 제외)
- **단락 수**: 7~12개. 한 단락 2~6줄, 줄바꿈 잦음.
- **표준 골격**(거의 모든 후기 글이 따름):
  1. **상단 매장 박스** — 글 본문 시작 직전:
     ```
     컬컴 하남점
     경기도 하남시 미사강변중앙로204번길 22 마이움센트럴아케이드 201호, 202호
     ```
  2. **도입 (계기·상황)** — "안녕하세요!" 또는 날씨/시기 후크 → 컬컴을 시작한 계기 1~2단락.
  3. **첫날·시작 후기** — 처음 와서 어땠는지 (긴장, 상담, 첫 스터디 느낌).
  4. **N개월/연차 후기** — 일정 기간 다닌 뒤 어떻게 달라졌는지(말이 트인다, 자신감, 일·여행에서 효과).
  5. **(선택) 오늘의 표현 / 오늘의 주제** — 짧은 패턴 3~5개를 인용. 단 학습 콘텐츠가 메인이 되면 안 됨, 후기 안 양념으로만.
  6. **(선택) 번호 리스트** — "이런 분들께 추천" / "달라진 점" / "1. 2. 3." 3~5개.
  7. **현장 분위기·스터디 방식** — 4~8명 소수그룹, 레벨 1·2 교재 / 레벨 3·4 매일 다른 프린트, "카페·LP샵 같은 분위기", 리더님·멤버분들과의 관계, 스터디 후 식사·카페 투어 등.
  8. **위치·접근성 (자주 포함)** — 미사역 도보 3분, 주차 가능, 강동·다산·감일에서도 옴.
  9. **CTA** — "👇 컬컴 스터디 알아보기👇" + 카카오톡 링크 임베드 + "편하게 문의해 보세요" 한 줄.
  10. **하단 마무리** — "저는 그럼 이만...!👋🏼" 같은 가벼운 끝인사 → 해시태그 한 줄.

이미지 위치는 본문 안에 `[이미지: 오늘의 스터디 현장]` 형태의 한 줄 가이드로만 표시 (운영자가 보고 직접 배치).

## 3. 제목 패턴

다음 중 하나의 형태:
- **솔직 후기형**: `하남에서 영어회화 찾고 계시다면? 내돈내산 컬컴 하남 솔직 후기`, `영어 울렁증 극복한 영어회화 스터디 왕초보 솔직 후기 컬컴 하남`
- **연차/도전기형**: `해외영업 직장인의 영어회화 스터디 도전기｜9개월 후 변화 솔직 후기 (컬컴 회(사)돈내산)`, `영어회화 프리토킹까지의 여정 컬컴 하남 스터디 1년 차 솔직 후기 하남 영어 학원 추천`, `abcd도 몰랐던 40대의 영어회화 도전기 | 컬컴 하남 성인영어회화`
- **체험·이벤트형**: `컬컴 하남 외국인 친구와 첫 벚꽃 데이트 다녀온 썰 | 성인 영어회화 스터디`
- **페르소나형**: `미사맘카페 성인영어회화 학원 추천 컬컴 하남 영유맘 솔직 후기`, `성인영어회화 컬컴 하남에서 영어공부하는 7세 아이 엄마 솔직후기`
- **조교 시점형**: `영어회화 스터디 컬컴 하남 오전 조교의 솔직 담백 후기 | 하남 강동 덕소 영어회화 컬컴 하남점 컬컴 리더`
- **여행·테마형**: `여행 영어회화 잘 하는 법｜컬컴 하남 스터디 후기 | 하남영어학원 하남비즈니스영어회화`

규칙:
- 25~50자
- "하남" 또는 "컬컴 하남" 키워드 포함 (지점 한정 강조)
- 구분자 `｜` 또는 `|` 또는 `(...)`로 SEO 키워드 1~3개 추가
- "잠실", "분당" 같은 다른 지점명 절대 금지

## 4. 톤 / 문체

- 부드러운 존댓말 (~해요, ~이에요, ~답니다, ~더라구요, ~네요).
- 1인칭 ("저는", "제가", "저도"), 회원 또는 조교 시점.
- 자주 쓰는 어미: `~더라고요`, `~었답니다`, `~한답니다`, `~네용`, `~네요ㅎㅎ`.
- 의성·감탄: `ㅎㅎ`, `ㅠㅠ`, `ㅋㅅㅋ`, `~~~`, `!!`, `....`, `:)`.
- 이모지: 🍀 ⭐️ 👇 😊 🙂 ✈️ 📚 ✏️ 🌸 🫶🏻 💕 📸 👋🏼. 글당 5~12개.
- 약점·고민·실패담을 솔직하게 드러내는 톤이 핵심. 자랑 톤 X.

## 5. 자주 쓰는 어휘·관용 표현

- `스터디` (강의·수업 대신 거의 항상 이 표현)
- `리더님`, `멤버분들`, `조교/조교님`
- `4~8명 소수그룹`
- `레벨 1·2 / 레벨 3·4`
- `매일 달라지는 프린트`
- `프리토킹`
- `카페 / LP샵 분위기`
- `딱딱한 학원 분위기가 아닌`
- `틀려도 괜찮은 환경`
- `말이 트이다 / 영어가 트이다`
- `으쌰으쌰`, `화목`, `소통`, `자극`, `시야가 넓어진다`
- `또 추가로 궁금한건 아래로 물어보면 다 답해주더라구요!`
- `👇 컬컴 스터디 알아보기👇`

## 6. CTA·매장·연락 정보

- 카카오톡 링크: `https://pf-link.kakao.com/qr/_xeMISK/pages/_Hxh?query=state=hanam,wsblog`
  - **반드시 `state=hanam,wsblog`** — 다른 지점 query 절대 금지.
- 매장 박스(상·하단 둘 다 가능):
  ```
  컬컴 하남점
  경기도 하남시 미사강변중앙로204번길 22 마이움센트럴아케이드 201호, 202호
  ```
- 위치 한 줄: `미사역 도보 3분`, `주차 가능`, `강동·다산·감일에서도 자주 오심`
- 전화: 운영자 메모에 명시된 경우만 인용 (없으면 카카오톡 링크만).

## 7. 해시태그 (마지막 줄)

본문 마지막 줄에 한 줄로 30~35개. 다음 풀에서 거의 그대로 사용:

```
#하남영어회화 #하남영어학원 #하남영어회화학원 #미사역역영어회화 #덕소영어회화 #하남역영어회화학원 #하남영어왕초보 #하남영어프리토킹 #하남영어 #하남직장인영어 #강동영어회화 #고덕영어회화 #상일동영어회화 #길동영어회화 #남양주영어회화 #남양주영어 #양평영어회화 #영어회화 #영어회화스터디 #하남맛집 #하남카페 #컬컴 #컬컴후기 #컬컴비용 #컬컴가격 #서이추 #서이추환영 #이웃 #이웃환영 #후기업 #미사술집 #미사맛집 #미사애견카페 #미사카페 #미사당근
```

원본 글은 `#하남영어회화` 사이에 공백 없이 붙여 쓴 형태(`#하남영어회화 #하남영어학원#하남영어회화학원`)도 자주 보이지만, 출력은 공백으로 분리한 위 형태로 통일한다.

## 8. 금지·주의

- **다른 지점명 (잠실·분당) 언급 금지**. 매장 박스 / 주소 / 카카오 query / 본문 어디에도.
- **학습 콘텐츠 단독 글 X**. 표현 강의가 본문의 50% 이상이면 안 됨. "오늘의 표현"은 양념으로 짧게만.
- 학생 실명·식별 가능한 외모 묘사·얼굴 식별 정보 금지. 사진 속 인물은 "멤버분", "리더님", "외국인 친구" 등으로만.
- 사진/메모에 없는 점수·합격·등록자 수 만들지 않음.
- "지금 등록 시 할인", "선착순 할인" 같은 자극 카피 금지.
- 트렌드 키워드(`trends.json`)는 본문에 1~2개만 자연스럽게 녹임. 키워드 나열·억지 삽입 금지. 충돌 시 후기 톤 우선.

## 9. 출력 형식

- 마크다운 코드블록(```)으로 감싸지 않는다.
- 네이버 스마트에디터에 그대로 붙여넣을 수 있는 평문.
- 단락 사이 빈 줄 1개. 단락 안은 일반 줄바꿈.
- 매장 박스는 별도 빈 줄로 명확히 구분.
- 마지막 줄은 해시태그 한 줄(공백 구분).
