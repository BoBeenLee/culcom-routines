# 인스타그램 스타일 가이드 — 컬컴 하남 (스터디 현황 / 후기)

레퍼런스: prompts/samples/ig/*.md (4개) — 모두 컬컴 **하남점** 정규 게시물이며, **스터디 현황** 카테고리에 한정.

이 가이드의 범위는 **하남점 한정**, **스터디 현황·후기 캡션** 한정이다. 다음은 다루지 않는다:
- 다른 지점
- 표현 학습 릴스(영어 패턴, 일본어 패턴) 단독 콘텐츠 — 별도 가이드 영역
- 일반 홍보 릴스(웰컴 영상, 매장 소개)

## 1. 게시물 종류

이 가이드는 **정규 게시물 (`/p/...`)** 형태의 스터디 현황 캡션을 대상으로 한다. 일상 스냅 + 짧은 캡션 + 매장 안내 + 해시태그 구조.

릴스 캡션은 이 가이드로 만들지 않는다. (운영자 메모에 "릴스" 명시가 있으면 본문만 1~3 문장으로 간단히 만들고 해시태그 생략.)

## 2. 캡션 본문

### 길이
- **본문 60~140자** (매장 안내·해시태그 제외).
- 1~3 문장. 대부분 한 문장이 길게 이어지고 줄바꿈으로 끊는다.

### 시작 패턴
다음 류의 톤으로 시작 (아래 예시는 sample 빈출 패턴 — 그대로 베끼지 말고 사진·시기에 맞는 변형으로):
- 친근한 호명: `멤버분들~!`, `안녕하세요~`
- 날씨/요일/시기 인사 (예시): `이번 주말 날씨가 좋네요~ 오늘도 열심히 스터디하시는 멤버분들!!`, `즐거운 일요일 다들 어떻게 보내고 계시나요~?`, `오늘은 비가 와서 날이 꾸리구리 했는데, 다들 비오는 날씨는 좋아하시나요?`
- 직전 이벤트 회상 (예시): `어제 어린이날 어디 다녀오셨나요~?`

### 본문 톤
- 따뜻한 존댓말 + 가벼운 끝맺음 (`~답니다`, `~네요`, `~죠`, `최고!!!`).
- "멤버분들" 호칭은 거의 모든 캡션에 등장.
- 학습 정보·문법 설명 X (이건 학습 릴스 영역).

아래는 sample 빈출 정형 문장이다. **이대로 베끼지 말 것** — 같은 문장이 매 글에 반복되면 캡션들이 같은 사람의 자동완성처럼 보인다. 사진·시기·메모에서 길어 올린 그 글만의 한 줄로 변형해 쓸 것.
- `오늘도 열심히 스터디하시는 멤버분들!! 항상 열정적인 모습 너무 멋있습니다`
- `주말 오전부터 컬컴 하남점은 즐거운 스터디 분위기!! 다들 놀러오세요~~~`
- `꿀꿀한 날씨에도 스터디 오신 영어/일본어 멤버분들 최고!!!`
- `영어회화/일본어회화 스터디를 가득 채웠답니다~~`

### 이모지
- 캡션 본문에 1~3개. 강조 단어 뒤에 1개 + 마지막 문장 끝에 1개.
- 자주 쓰이는: 😊 🍀 ❤️ 😍 👍 ✨ 🫶.

## 3. 매장 안내 라인 (캡션 본문과 해시태그 사이)

본문 → 빈 줄 → 다음 두 줄 → 빈 줄 → 해시태그 한 줄. 두 줄은 거의 그대로 인용:

```
🔗스터디 분위기가 궁금하시다면 언제든 프로필 링크로❤️
010-2946-0553 전화도 언제나 열려있습니다~!
```

생략하는 경우는 거의 없다. 운영자 메모에 "안내 빼고" 같은 명시 요청이 있을 때만 제거.

## 4. 해시태그 (한 줄)

게시물 주제에 따라 두 풀 중 하나를 선택. 한 줄에 모아 작성하고 캡션과는 빈 줄 1개로 구분.

### A. 영어+일본어 모두 다루는 일상 스터디 현황 (24개 풀, 가장 자주)


기본은 **A**. 사진/메모가 명백히 영어 스터디만 다룰 때만 **B**.

트렌드 키워드(`trends.ig_hashtags`)는 풀과 중복 제거 후 1~3개만 추가. 풀과 충돌 시 풀 우선.

## 5. 매장·연락 정보 (사실 고정값)

- 상호: 컬컴 하남점 (정식: 하남 영어회화/일본어회화 컬컴 하남점)
- 위치: 5호선 미사역 2번출구 도보 3분, 주차 가능
- 전화: 010-2946-0553
- 프로필 링크 카카오톡: `pf-link.kakao.com/qr/_xeMISK/pages/_Hxh?query=state=hanam,instanoad,1`
- 약력 한 줄: `💘23년차 평생교육원 영어/일본어회화 컬컴 하남점`

운영자 메모에 다른 번호가 들어오면 그걸 우선.

## 6. 금지·주의

- **다른 지점(잠실·분당) 언급 금지**.
- 학습 콘텐츠(문법, 패턴 강의) 캡션 X — 이 가이드 영역 아님.
- 학생 실명·식별 가능한 외모 묘사·얼굴 식별 정보 금지. 사진 속 인물은 "멤버분들", "리더님" 등 일반 표현.
- 사진에 없는 사실(예: "오늘 신규 5명 등록") 만들지 않음.
- "할인 이벤트", "지금 등록 시 무료" 같은 자극 카피 금지.

## 7. 출력 형식

- 마크다운 코드블록 없이 평문.
- 구조: `[본문 1~3문장]` → 빈 줄 → `[매장 안내 두 줄]` → 빈 줄 → `[해시태그 한 줄]`
- 운영자가 인스타 앱에 그대로 붙여넣으면 즉시 게시 가능한 상태.
