# 네이버 블로그 스타일 가이드 — 컬컴 하남 (스터디 후기 / 현황)

레퍼런스: prompts/samples/naver/*.md (10개) — 모두 컬컴 **하남점** 글이며, **스터디 후기 / 스터디 현황** 카테고리에 한정.

이 가이드의 범위는 **하남점 한정**, **후기·체험·스터디 현황** 한정이다. 다음은 다루지 않는다:
- 다른 지점 (잠실·분당)
- 학습 콘텐츠(표현 강의, 문법, 회화 팁) 단독 글
- 이벤트·플리마켓 단독 안내 글

## 1. 글의 정체성

- **장르**: 1인칭 후기·체험기·스터디 현황 일기.
- **화자(페르소나)**: 다음 중 사진과 메모에 가장 잘 맞는 하나를 잡는다.
  - 직장인 회원 (해외영업·법카·바쁜 직장인) — 예: `samples/naver/224249814155.md`
  - 아이 엄마 / 영유맘 / 7세 학부모 — 예: `224259930834.md`, `224262618982.md`
  - 왕초보 / 영어 울렁증 (40대 abcd 도전기 포함) — 예: `224273780353.md`, `224274652408.md`
  - 장기 회원 (6개월~1년차) "달라진 점" / 프리토킹 도달기 — 예: `224254535128.md`, `224265989671.md`
  - 컬컴 조교 시점 ("스터디 분위기를 이끌고 있어요") — 예: `224267970727.md`
  - 외국인 친구·데이트·계절 테마형 (체험 일기 + 학습 양념) — 예: `224254535128.md`
  - 여행/비즈니스 영어 테마형 (테마 후크 + 후기) — 예: `224277603205.md`
- **목표**: 컬컴 하남의 분위기를 1인칭으로 보여주고, 마지막에 카카오톡 문의로 자연스럽게 유도.
- **페르소나 선택 규칙 (필수)**: 운영자 메모(`subject`/`vibe`)가 비어 있거나 모호해도, 사진·트렌드를 종합해 위 7개 페르소나 중 **하나만 명시적으로 선택**하고 글 **전체**(후크·동기·중반 에피소드·결론·CTA까지) 에서 그 페르소나의 시점·동기·어휘·어미를 일관되게 유지한다. 후크 한두 단락만 페르소나에 맞추고 본문이 "일반 회원 1인칭"으로 회귀하는 패턴은 동질화의 주된 원인 — 피한다.

## 2. 길이와 골격

분량은 **첨부된 이미지 수**에 따라 가변. 워크플로우의 draft.mjs가 이미지 수 기반 가이드를 함께 주입하므로 그것을 우선 따르되, 실측 기반 기준은 다음과 같다 (10개 reference 샘플 본문 character count 측정).

| 이미지 수 | 본문 분량 (자) | 단락 수 | 구조 |
| --- | --- | --- | --- |
| 1~3장 | 1,200~3,000 | 5~10 | 한 가지 인상·에피소드에 집중하는 짧은 후기 |
| 4~9장 | 1,700~4,500 | 8~14 | 표준 후기 골격, 섹션 2~4개 |
| 10~14장 | 2,800~5,500 | 12~18 | 페르소나 동기 + 학습 양념 또는 후기 결론 풍성 |
| 15장+ | 3,500~5,500 | 14~20 | 테마 후크 + 본문 + 위치 안내까지 다 포함 |

- 한 단락 2~6줄. **줄바꿈 잦음** — 한 문장씩 끊어서 별도 줄로 쓰는 패턴이 흔함 (모바일 가독성).
- **한 줄당 24자 이내 (mobile 기준)** — 본문 한 줄에 글자 수가 24를 넘기지 않도록 어절·구 단위로 줄바꿈. 25자 이상 한 줄에 몰아쓰지 말 것. 위 "줄바꿈 잦음" 패턴을 한 단계 더 강하게 적용한다고 보면 된다. 예외: 위젯 chrome 한 줄 (매장 박스 / 지도 / 카카오) 과 마지막 해시태그 한 줄은 single-line 유지가 우선이라 24자 제한 적용 안 함.
- 단락 사이 빈 줄에 zero-width space (`​`) 가 들어간 줄이 흔하지만 mtnb 가 알아서 처리하므로 draft 는 일반 빈 줄로 둬도 무방.
- 본문 안 이미지 위치는 `[이미지 #N: 짧은 묘사]` 한 줄로 표시. 첨부된 모든 사진(N=1..imageCount)이 본문 어딘가에 정확히 한 번씩 등장해야 함.
  - 예: `[이미지 #1: 멤버분들이 둘러앉아 발표 자료 보는 모습]`
  - 마크다운 `![](url)` 문법은 직접 쓰지 말 것. workflow가 마커 뒤에 자동으로 `![](원본 URL)`을 주입해 GitHub 코멘트에서 사진이 인라인으로 보이게 한다. 운영자는 네이버 에디터에 붙여넣을 때 `![]()` 줄만 지우고 같은 위치에 사진을 직접 끌어 올린다.
- **위젯 chrome — single-line raw 형태로 직접 출력**: 매장/지도/카카오 카드는 다음 정확한 single-line raw chrome 형태로 본문에 출력 (운영자가 그대로 붙여넣었을 때 네이버 에디터가 카드로 렌더링).
  - 매장 박스 (예: `224241400842.md` 10줄):
    `[**컬컴 하남점**경기도 하남시 미사강변중앙로204번길 22 마이움센트럴아케이드 201호, 202호](#)`
  - 네이버 지도 위젯 (예: `224249814155.md`):
    `[**네이버지도**컬컴 하남점map.naver.com](https://map.naver.com/p/entry/place/1965780250)`
  - 카카오 채널 링크 (모든 샘플 CTA 끝):
    `[**카카오톡**pf-link.kakao.com](https://pf-link.kakao.com/qr/_xeMISK/pages/_Hxh?query=state=hanam,blog)`
- 사진의 분위기·계절감·장소·소품·인물 활동을 본문에 자연스럽게 녹임. 사진에서 보이지 않는 사실 만들지 말 것.

### 참고 블록 풀 (이미지 4장 이상 — 후기 글에서 자주 관찰되는 빈출 블록)

아래는 **이 글에 모두 다 등장해야 하는 단일 표준 골격이 아니라**, 10개 reference 샘플에서 추출한 **블록 풀**이다. 페르소나·메모·사진에 맞춰 일부 블록만 골라 다른 순서로 조합한다. 같은 풀을 매번 같은 순서로 채우면 글이 동질화되니, 매 글마다 **선택하는 블록·순서·강조 비중을 의도적으로 다르게** 가져간다.

필수: 1, 2 (오프닝 중 하나), 8, 10. 그 외는 글 성격에 맞춰 4~7개 선택.

1. **타이틀** (H1, 한 줄) — 필수
2. **오프닝** — 다음 패턴 중 하나만 선택 (필수):
   - **(A) 매장 박스 시작**: `[이미지 #1: 매장 위치 지도]` + 매장 박스 raw chrome (예: `224241400842.md` 9~12줄, `224274652408.md`)
   - **(B) 후크 시작**: 인용블록 또는 한 줄 후크 + `[이미지 #1: ...]` (예: `224249814155.md`, `224277603205.md`)
   - **(C) 시기·날씨·이벤트 인사**: "벌써 5월", "비 오는 주말", "어제 어린이날" 같은 톤 후크로 시작 (예: `224262618982.md`)
3. **페르소나 동기** — 1~3 단락. "왜 컬컴을 시작했는가". 선택한 페르소나에 맞는 단 한 가지 동기만 깊게 다룰 것 (회사 법카, 아이 영어 함께, 여행 답답함, 울렁증 극복, 외국인 친구 등 중 하나). 동기를 여러 개 나열하면 글이 일반화됨.
4. **본문 섹션** (1~4 개, 글 분량에 맞게 가변) — 다음 헤더 패턴 중 하나만 글 전체에서 일관되게 사용 (섹션마다 다른 패턴 섞지 말 것):
   - **인용블록 헤더**: `> **첫날 후기**`, `> **컬컴에 온 계기**`
   - **번호+Bold**: `**1. 오늘의 표현으로 대화문 만들기**`
   - **이모지+Bold**: `**⭐️컬컴 하남점 스터디 현장 분위기⭐️**`
   - 각 섹션 안에 사진을 1~3장 인터리브. 사진 묘사를 텍스트로 다시 풀어쓰지 말 것.
5. **(선택) 결론·추천 대상** — "이런 분들께 추천", "달라진 점 N가지" 같은 마무리. 모든 글에 다 넣지 않음 — 절반 정도만.
6. **(선택) 현장 분위기·스터디 방식** — 4~8명 소수그룹, 레벨별 교재, 카페·LP샵 분위기, 리더님·멤버분들. **이 단락을 모든 글에 다 넣지 말 것** — 글의 절반에만 등장. 넣을 때도 위 키워드 중 2~3개만 골라 자연스럽게 흘리고, 모두 나열하지 않음.
7. **(선택) 위치·접근성** — 미사역 도보 3분, 주차 가능, 강동·다산·감일. 글마다 들어가는 키워드 부분집합을 다르게.
8. **CTA** — `**👇 컬컴 스터디 알아보기👇**` 한 줄 + `[이미지 #N: 카카오 채널 안내 이미지]` + 카카오 채널 raw chrome (필수).
9. **(선택) 끝인사** — `**저는 그럼 이만...!👋🏼**` 같은 한 줄. 모든 글에 매번 같은 문구로 넣으면 동질화 — 절반 정도만, 혹은 다른 가벼운 끝인사로 바꿔 가며.
10. **해시태그 본문** — 마지막 줄 (필수, §7 의 풀에서 sampling — 거기 규칙 따름).

### 동질화 방지 (필수)

같은 운영자 메모·비슷한 이미지로 여러 글을 생성할 때 다음을 의식적으로 변형하라:
- **오프닝 패턴**: 위 (A/B/C) 중 직전 글과 다른 것을 우선 선택.
- **섹션 헤더 스타일**: 인용블록 / 번호Bold / 이모지Bold 중 직전 글과 다른 것을 선택.
- **선택 블록 (5/6/7/9)**: 매번 다른 부분집합을 사용 — 모든 블록을 매 글마다 다 넣지 말 것.
- **반복 phrase (§5 의 "재등장 자제 표현")**: §5 에서 명시한 표현은 새 글에서 1 개 이하로만 등장시킬 것. 대체 표현을 사진·메모에서 직접 길어 올려 쓴다.
- **해시태그 (§7)**: 매 글마다 다른 부분집합·다른 순서.

### 이미지 적은 경우 (1~3 장)

핵심 4~6 블록만 선택. 매장 박스, 페르소나 동기, 한 가지 에피소드, CTA, 해시태그.

## 3. 제목 패턴

다음 중 하나의 형태:
- **솔직 후기형**: `하남에서 영어회화 찾고 계시다면? 내돈내산 컬컴 하남 솔직 후기`, `영어 울렁증 극복한 영어회화 스터디 왕초보 솔직 후기 컬컴 하남`
- **연차/도전기형**: `해외영업 직장인의 영어회화 스터디 도전기｜9개월 후 변화 솔직 후기 (컬컴 회(사)돈내산)`, `영어회화 프리토킹까지의 여정 컬컴 하남 스터디 1년 차 솔직 후기 하남 영어 학원 추천`, `abcd도 몰랐던 40대의 영어회화 도전기 | 컬컴 하남 성인영어회화`
- **체험·이벤트형**: `컬컴 하남 외국인 친구와 첫 벚꽃 데이트 다녀온 썰 | 성인 영어회화 스터디`
- **페르소나형**: `미사맘카페 성인영어회화 학원 추천 컬컴 하남 영유맘 솔직 후기`, `성인영어회화 컬컴 하남에서 영어공부하는 7세 아이 엄마 솔직후기`
- **조교 시점형**: `영어회화 스터디 컬컴 하남 오전 조교의 솔직 담백 후기 | 하남 강동 덕소 영어회화 컬컴 하남점 컬컴 리더`
- **여행·테마형**: `여행 영어회화 잘 하는 법｜컬컴 하남 스터디 후기 | 하남영어학원 하남비즈니스영어회화`

규칙:
- 25~50자
- "하남" 또는 "컬컴 하남" 키워드 포함 (지점 한정 강조)
- 구분자 `｜` 또는 `|` 또는 `(...)`로 SEO 키워드 1~3개 추가
- "잠실", "분당" 같은 다른 지점명 절대 금지

## 4. 톤 / 문체

- 부드러운 존댓말 (~해요, ~이에요, ~답니다, ~더라구요, ~네요) 기조 유지.
- 1인칭 ("저는", "제가", "저도"), 회원 또는 조교 시점.
- 어미·의성·감탄·이모지는 sample 글 (`prompts/samples/naver/*.md`) 참고. 글의 톤·페르소나에 맞게 자유롭게. 같은 어미/이모지 풀을 매 글마다 같은 비율로 쓰면 글들이 같은 사람 글처럼 보이니, sample 안에서도 어떤 글은 이모지를 많이, 어떤 글은 거의 안 쓰는 식의 폭을 의식적으로 가져갈 것.
- 약점·고민·실패담을 솔직하게 드러내는 톤이 핵심. 자랑 톤 X.

## 5. 어휘·관용 표현

핵심 (거의 항상 등장 — 자연스럽게 쓸 것):
- `스터디` (강의·수업 대신 거의 항상 이 표현)
- `리더님`, `멤버분들`, `조교/조교님`

재등장 자제 표현 (과거 글에서 반복적으로 등장 — 새 글에서 1 개 이하만, 자연스럽게 자리잡힐 때에만 사용):
- `4~8 명 소수그룹`
- `레벨 1·2 / 레벨 3·4`
- `매일 달라지는 프린트`
- `프리토킹`
- `카페 / LP샵 분위기`, `딱딱한 학원 분위기가 아닌`
- `틀려도 괜찮은 환경`
- `말이 트이다 / 영어가 트이다`
- `으쌰으쌰`, `화목`, `소통`, `자극`, `시야가 넓어진다`
- `또 추가로 궁금한건 아래로 물어보면 다 답해주더라구요!`

→ 위 표현들은 sample 글마다 단골이라 매 글에 다 모이면 글들이 서로 구분되지 않는다. 분위기·인테리어·스터디 방식·환경을 묘사할 때는 위 phrase 대신 **사진과 운영자 메모에서 직접 길어 올린 디테일** (그 글만의 단서) 로 쓰는 것을 우선한다.

CTA 정형 라인 (예외 — CTA 직전에 거의 항상 사용):
- `👇 컬컴 스터디 알아보기👇`

## 7. 해시태그 (마지막 줄)

## 8. 금지·주의

- **다른 지점명 (잠실·분당) 언급 금지**. 매장 박스 / 주소 / 카카오 query / 본문 어디에도.
- **학습 콘텐츠 단독 글 X**. 표현 강의가 본문의 50% 이상이면 안 됨. "오늘의 표현"은 양념으로 짧게만.
- **위젯 chrome 정확한 형식 준수** — §2 / §6 의 canonical single-line raw chrome 과 글자 단위로 동일하게 출력 (운영자가 그대로 붙여넣었을 때 카드로 렌더되는 형태).
  - 카카오 query 는 반드시 `state=hanam,blog`. 다른 값 (`,wsblog`, `,instanoad`, `,instanoad,1`, 다른 지점 prefix 등) 금지.
  - 매장 카드 link target 은 `(#)`, 지도 카드는 `(https://map.naver.com/p/entry/place/1965780250)`. URL 임의로 바꾸지 말 것.
  - 줄바꿈 절대 금지 — 모든 chrome 은 `[**라벨**컨텐츠...](url)` single-line 한 줄.
  - 자유 형식 (plain `[카카오톡](url)`, 라벨 없이 등) 금지.
- 학생 실명·식별 가능한 외모 묘사·얼굴 식별 정보 금지. 사진 속 인물은 "멤버분", "리더님", "외국인 친구", "조교님" 등으로만.
- 사진/메모에 없는 점수·합격·등록자 수 만들지 않음.
- "지금 등록 시 할인", "선착순 할인" 같은 자극 카피 금지.
- 트렌드 키워드(`trends.json`)는 본문에 1~2개만 자연스럽게 녹임. 키워드 나열·억지 삽입 금지. 충돌 시 후기 톤 우선.
- 모델이 가끔 출력하는 도구호출 누출 (`update_topic(...)`, `<ctrl##>`, `strategic_intent:` 등) 절대 출력 금지 — 운영자 코멘트에 그대로 노출되면 안 됨.

## 9. 출력 형식 (mtnb 호환 마크다운)

`@jjlabsio/md-to-naver-blog` 라이브러리가 워크플로우에서 마크다운을 네이버 호환 HTML로 변환한다. 다음 마크다운 구조로 출력하라.

```
# 글 제목 (H1, 한 줄)

본문 첫 단락. 평소 톤대로.

## 소제목 1 (필요 시)

본문 단락...

[이미지 #1: 짧은 묘사]

본문 다음 단락...

- 리스트 항목
- 리스트 항목

## 소제목 2

...

#하남영어회화 #하남영어학원 #컬컴 ...
```

규칙:
- **첫 줄은 반드시 `# ` 으로 시작하는 H1 — 글 제목**. mtnb 가 이 H1을 제목으로 추출한다.
- 코드블록(```)으로 전체를 감싸지 말 것.
- 섹션 분할이 필요하면 `## ` H2 사용. H1은 글 제목 한 번만.
- 강조: `**굵게**`, `_기울임_`, `> 인용` 자유롭게.
- 리스트: `- 항목` 또는 `1. 항목`.
- 이미지 위치는 `[이미지 #N: 묘사]` 마커 한 줄 (workflow가 `![alt](url)` 로 자동 변환).
- 마지막 줄은 해시태그 한 줄(공백 구분).
- 컬컴 매장 박스(주소)는 본문 단락으로 자연스럽게 작성하거나 인용 블록(`> `) 사용.
